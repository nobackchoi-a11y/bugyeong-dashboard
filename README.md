# 부경사업단 업무앱

GitHub + Netlify 자동배포용 구조입니다.

## 파일 구조

- index.html
- manifest.json
- dashboards/activity.html
- dashboards/agency.html
- dashboards/manager-image.html

## Netlify 연결

1. GitHub에 새 Repository 생성
2. 이 폴더 안의 파일 전체 업로드
3. Netlify > Add new project > Import from GitHub
4. 해당 Repository 선택
5. Build command 비워두기
6. Publish directory: ./
7. Deploy
